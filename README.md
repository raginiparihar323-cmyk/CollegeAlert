# College Alert 📚🔔

## Smart Campus Event Reminder Android Application

College Alert is a modern Android application developed using Kotlin and Firebase to simplify campus event management and communication between administrators and students. The application provides a centralized platform where students can stay informed about important college activities, while administrators can efficiently manage and publish campus updates.

---

## Project Objective

The primary objective of this application is to ensure that students never miss important academic or extracurricular events by providing a single platform for viewing and managing campus announcements.

The application covers:

* Seminars
* Workshops
* Examinations
* Fests
* Important Notices

---

## Key Features

### Administrator Module

* Secure Admin Login
* Add New Events
* Edit Existing Events
* Delete Events
* Manage Campus Updates
* View Event Statistics Dashboard

### Student Module

* Student Registration & Login
* View Upcoming Events
* Search Events Instantly
* View Detailed Event Information
* Add Events to Device Calendar
* Receive Event Reminder Notifications
* Access Personal Profile

---

## Event Management System

The application allows administrators to create and organize different categories of campus events. Every event contains:

* Event Title
* Date
* Time
* Venue
* Category
* Description

Students can easily access all event information through a clean and user-friendly interface.

---

## Technologies Used

* Kotlin
* Android Studio
* Firebase Authentication
* Firebase Realtime Database
* WorkManager
* XML UI Design
* Material Design Components

---

## System Architecture

1. Administrator creates and manages events.
2. Event data is stored in Firebase Realtime Database.
3. Students access event information through the application.
4. Notifications and reminders help students stay updated.
5. Calendar integration allows students to save important events directly to their devices.
